# Git

## Git介绍

Git是一个**分布式版本控制系统**，最初目的是为更好地管理Linux内核开发而设计  
Git和其他**集中式版本控制系统**不一样，不需要服务器端软件，就可以运作版本控制，**速度快得多**，每个人的电脑上都是一个完整的版本库，直接在本机上获取数据，不必连接到主机端获取数据  
**Git还有强大的分支合并管理功能**
